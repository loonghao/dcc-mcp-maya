# maya-scripting

Maya scripting skill. Provides actions for executing MEL and Python code directly inside Maya's interpreter.

## Scripts

- `execute_mel` — Execute a MEL script inside Maya
- `execute_python` — Execute Python code inside Maya's interpreter
